# XingJu v2.0 - 快速测试指南

## 🚀 5 分钟快速测试

### 步骤 1: 安装依赖 (2 分钟)

```bash
cd XingJu-v2
npm install
```

### 步骤 2: 启动开发服务器 (1 分钟)

```bash
# 方式 1: Vite 前端预览 (推荐)
npm run dev

# 方式 2: Tauri 桌面应用
npm run tauri dev
```

### 步骤 3: 访问应用 (1 分钟)

浏览器打开：`http://localhost:5173`

### 步骤 4: 功能验证 (1 分钟)

- [ ] 点击侧边栏切换 4 个模块
- [ ] 在搜索框输入文字

- [ ] 点击播放按钮

## ⚠️ 常见问题

### 问题 1: 样式不生效

**解决:**
```bash
# 清除缓存重启
rm -rf node_modules package-lock.json
npm install
```

### 问题 2: 动画卡顿

**解决:**
- 关闭浏览器硬件加速
- 减少浏览器标签页数量
- 检查 GPU 占用

### 问题 3: 模块导入错误

**解决:**
```bash
# 检查文件路径
ls -la src/components/
ls -la src/styles/
```
