## ✨ 重大更新

### 🎨 全新设计系统

**高级赛博朋克风格** - 在保留赛博朋克核心元素的基础上，注入精致、现代的设计语言。

#### 核心组件
- **玻璃态面板** - 半透明背景 + 背景模糊 + 顶部高光
- **渐变文字** - 青→紫→粉三色渐变
- **霓虹边框** - 渐变边框 + 悬停光晕
- **赛博按钮** - 光泽扫过动画 + 悬停效果
- **徽章系统** - 多颜色变体 (青/洋红/紫)

#### 动画系统
- **fade-in-up** - 元素进入动画
- **fade-in-scale** - 缩放进入
- **shimmer** - 加载骨架屏
- **pulse-glow** - 呼吸光晕
- **rotate-slow** - 缓慢旋转
- **float-particle** - 粒子浮动

---

### 🌟 多层背景系统

1. **底层** - 虚空黑背景 (#050508)
2. **渐变层** - 三层径向渐变光晕
3. **网格层** - 赛博网格叠加
4. **粒子层** - 20 个浮动粒子

## 📋 升级指南

### 从 v1.x 升级

```bash
# 1. 备份数据
cp -r ~/.xingju/data ~/xingju-backup

# 2. 下载新版本
# 见下方下载链接

# 3. 安装新版本
# Windows: 运行 MSI 安装包
# macOS: 拖拽 DMG 到 Applications
# Linux: sudo dpkg -i xingju_2.0.0_amd64.deb

# 4. 启动应用
# 数据会自动迁移
```

### Web 版本部署

```bash
# 1. 解压
tar -xzf xingju-v2.0.0-web.tar.gz

# 2. 使用 Nginx
sudo cp -r dist /var/www/xingju
sudo systemctl restart nginx

# 3. 或使用 Vercel/Netlify
# 直接上传 dist/ 目录
```
